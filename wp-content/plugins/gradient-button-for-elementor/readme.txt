Elementor Gradient Button is a simple gradient button addon for Elementor Page Builder.

== Description ==

<strong>Gradient Button for Elementor</strong>

Easy to use gradient button addon for Elementor Page Builder. No CSS coding required. Simply drag and drop the widget and start styling the button with predefined options.

== Installation ==

1. Install the plugin through the WordPress plugins screen or upload the plugin files to the ‘/wp-content/plugins/plugin-name’ directory.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Can I use Gradient Button for Elementor without Elementor? =

No.
The Gradient Button for Elementor is an addon for Elementor. You need Elementor to be activated.
